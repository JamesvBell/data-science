# How to Use This Template

1. Copy or clone this folder and rename it to your new project name.
2. Replace the `README.md` content with your project-specific summary.
3. Add your dataset to the `/data` folder.
4. Add your BRD or PRD to the `/requirements` folder.
5. Create a new Jupyter notebook inside `/notebooks` to begin working.

This structure helps keep your work clean, modular, and professional.
